Grundlæggende web

Da vi skulle starte på tema 2: Grundlæggende web fik jeg en grundlæggende forståelse for design og udvikling af brugergrænseflader til web. Jeg arbejdede med hele processen fra idé og design til implementering i HTML og CSS med fokus på brugervenlighed, struktur og teknisk korrekthed.

Vi fik en yderligere introduktion til Figma som designværktøj og lærte at udvikle wireframes og styletiles. I den forbindelse arbejdede vi med designkonventioner, herunder gestaltprincipper, typografi, farver, kontrast og tekstopsætning, for at sikre klare og brugervenlige brugeroplevelser. Jeg har også mere om at planlægge navigation, mappestruktur og navnekonventioner, så mine websites ville være mere overskuelige.

Derudover arbejdede vi med billedformater til web såsom SVG, WebP, PNG og JPG, herunder valg af format, billedstørrelse, beskæring og brug af metadata. Jeg fik introduceret ophavsret, licenser og rettigheder, så medier anvendes korrekt og lovligt.

På det tekniske plan lærte jeg at opbygge websites med HTML og CSS, med klar separation mellem struktur og styling, samt at validere kode. Vi arbejdede med responsivt design ud fra et mobile-first-princip og anvendte CSS Grid, Flexbox og media queries til at skabe fleksible layouts, der fungerer på tværs af skærmstørrelser.

Endelig har jeg brugt browserens developer tools til at teste, fejlfinde og optimere HTML, CSS og billedressourcer samt arbejdet med tilgængelighed gennem korrekt farvekontrast og læsbar typografi.
